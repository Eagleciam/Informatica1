Crash Elements

Para mover los cuadros :

Cuadro Naranja :

  • Presione tecla a para mover a la izquierda
  • Presione tecla d para mover a la derecha
  • Presione tecla s para mover a la arriba
  • Presione tecla w para mover a la abajo

Cuadro Azul :

  • Presione tecla j para mover a la izquierda
  • Presione tecla l para mover a la derecha
  • Presione tecla i para mover a la arriba
  • Presione tecla k para mover a la abajo
